Script OpcodesÂ¶

Note

Besides the Opcodes included in the following table, NavCoin Core supports also every other Opcode present in Bitcoin Core 0.13.0

Name Value Function
OP_CFUND 0xc1 DAO Special Scripts
OP_PROP 0xc2 DAO Special Scripts
OP_PREQ 0xc3 DAO Special Scripts
OP_YES 0xc4 DAO Special Scripts
OP_NO 0xc5 DAO Special Scripts
OP_ABSTAIN 0xc7 DAO Special Scripts
OP_REMOVE 0xc8 DAO Special Scripts
OP_DAO 0xc9 DAO Special Scripts
OP_ANSWER 0xca DAO Special Scripts
OP_CONSULTATION 0xcb DAO Special Scripts
OP_COINSTAKE 0xc6 Pushes TRUE to the stack if the output spent in a coinstake transaction, FALSE otherwise.
OP_POOL 0xd0 Used to signal a block has been staked by NavPool
